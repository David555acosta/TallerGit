# CV Web - David Acosta

Este proyecto es un CV técnico desarrollado en HTML y CSS.
El objetivo es presentar mi perfil profesional de manera clara,
simple y responsive.

---

## 📌 Descripción

Se trata de una página web estática donde presento:

- Mi perfil profesional
- Tecnologías que utilizo
- Proyectos destacados
- Información de contacto

El diseño está construido siguiendo buenas prácticas de HTML semántico y CSS responsive (Mobile First).

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- Responsive Design (Media Queries)
- Flexbox
- CSS Grid

---

## 🚀 Funcionalidades

- Presentación de perfil profesional
- Lista de tecnologías adaptable a distintas resoluciones
- Sección de proyectos con enlaces a GitHub
- Información de contacto
- Diseño responsive
